Investment Property Evaluator

This project is a web-based financial tool designed to help users evaluate the profitability of real estate investments. It allows users to input key variables such as purchase price, financing details, rental income, and expenses to generate important financial metrics.

Key Features

  • Monthly Cash Flow Calculation
  • Cash-on-Cash Return
  • Cap Rate Analysis
  • Amortization Schedule
  • Break-even Rent Calculator

Purpose

The goal of this project is to simplify real estate investment analysis by automating calculations that would normally require spreadsheets or financial calculators. This tool helps investors make faster and more informed decisions.
